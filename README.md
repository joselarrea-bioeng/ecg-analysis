# ECG Signal Analysis 🫀

Analysis of cardiac signals using real data from the MIT-BIH Arrhythmia Database.

## Description
This project analyzes ECG signals to study heart rate variability and detect anomalies such as arrhythmias, using Python and biomedical data science tools.

## Results
- Heart rate average: ~73 BPM
- Detected arrhythmia episode: 366 BPM outlier
- HRV (Heart Rate Variability) analysis

## Technologies
- Python 3.13
- Pandas, Numpy
- Matplotlib, Seaborn
- WFDB (PhysioNet)

## Data Source
MIT-BIH Arrhythmia Database - PhysioNet
